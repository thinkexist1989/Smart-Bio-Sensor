# Smart-Bio-Sensor

石墨烯生物传感器

- Hardware：硬件，包括了采集板、运放板、跳线版
- Firmware：固件，基于STM32CubeMX，采用HAL库
- Windows：Windows下的数据实时显示
- Android：Android端的数据实时采集
